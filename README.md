# Neo Desk

Neo Desk is a AI-powered personal workspace.
You can work with AI's colleagues in the office software like Slack or Lark.

## Features

- Workspace
- Chat
- Task
- Calendar
- Document
- AI Assistant

## Tech Stack

- Next.js
- TailwindCSS
- TypeScript
- Shadcn UI
- Framer Motion
- Better Auth
- Drizzle ORM
- PostgreSQL
- Hono
- Tanstack Query
- AI SDK
- Docker
- RAG

## Services

- Neon
- Upstash
